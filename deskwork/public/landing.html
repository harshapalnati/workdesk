Explicit approvals

Sensitive tools pause until the user replies with approve <action>.

Native shell

Tauri + Rust commands for file I/O, system stats, screenshots, and automation.

Bun toolchain

Fast install/build, zero Node bloat. Vite stays for the React layer.

Privacy defaults

API keys live in the keyring; tool outputs are redacted before sessions are stored.

Flow
01. Context
Agent reads working dir + active window (Windows) to scope actions.
02. Plan
LLM drafts steps; sensitive ones are flagged for approval.
03. Approve
User replies approve execute_command or approve all.
04. Act
Rust commands run; activity feed streams events; sessions persist without raw tool output.
approve execute_command
approve write_file
approve all
Safety posture
  • •Sensitive tools (shell, file writes, input simulation, web open) are gated by chat approval.
  • •Tool outputs and screenshots are redacted before session persistence.
  • •API keys stored in OS keyring; disk copy is blanked on save.
  • •Frontend surfaces pending approvals to prevent silent execution.
